# Amareswar Nayak - Developer Portfolio

A modern, animated developer portfolio website built with React.js, featuring a dark minimal futuristic theme with smooth animations and responsive design.

## Features

- **Dark Minimal Futuristic Theme**: Black background (#0a0a0a) with large bold typography
- **Smooth Animations**: CSS keyframes and transitions for engaging user experience
- **Responsive Design**: Optimized for mobile and desktop devices
- **Geometric Shapes**: Abstract circle shapes in background with floating animations
- **Scroll Progress Indicator**: Visual progress bar at the top
- **Custom Cursor**: Interactive cursor effects
- **Smooth Scrolling**: Seamless navigation between sections

## Sections

1. **Hero**: Large typography introduction with call-to-action buttons and social links
2. **About**: Personal journey and background information
3. **Skills**: Animated skill cards showcasing technical expertise
4. **Startup**: Information about LaunchXpax startup
5. **Projects**: Showcase of key projects
6. **Contact**: Contact form and professional links
7. **Footer**: Copyright and credits

## Tech Stack

- **Frontend**: React.js
- **Styling**: CSS3
- **Build Tool**: Vite
- **Animations**: Pure CSS

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Open [http://localhost:5173](http://localhost:5173) in your browser

## Build for Production

```bash
npm run build
```
